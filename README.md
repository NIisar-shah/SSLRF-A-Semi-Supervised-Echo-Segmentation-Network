# SSLRF-A-Semi-Supervised-Echo-Segmentation-Network
This repo is about echocardiographic segmentation using a semi-supervised network
Download the CAMUS and Echo-Net from their websites. https://www.creatis.insa-lyon.fr/Challenge/camus/, https://echonet.github.io/dynamic/index.html#dataset,
extract the images from echnet videos. The ratio of labelled and unlabel should be 20 and 80 percent using the Python file extract_echonet_frames.py.
